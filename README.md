# Parking-Control-API
Construção de um projeto Spring Boot (Parking Control API), contem os métodos POST, GETONE, GETALL, DELETE, PUT.

Preparação Ambiente:
- JDK 17
- Maven
- Postman 
- PgAdmin (Postgres)
- IDE (IntelliJ, STS, Eclipse, VsCode, etc)
